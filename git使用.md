*   创本地仓库前还是把远程仓库在github里面==先创好==。

*   git init

*   git pull  `拉取从远程仓库获取的资源`

*   然后git add .

*   git commit -m '注释'

*   `git remote add origin https://github.com/xxxx/xxx.git或者是ssh形式的url`用来创建远程版本库origin，并完成对接。

*   `git push -u origin master`本地传递数据到远程库==-u==为第一下上传需要以后直接`git push`。
